# Void10100Cmd

![Pack Icon](pack_icon.png)

Краткое описание: Этот аддон добавляет продвинутые команды для управления сервером.
Чтобы увидеть список всех команд, введите команду `!help`.

## 🚀 Установка
1. Скачайте актуальный `mcpack` из [Releases](../../releases).
2. Запустите файл, чтобы импортировать в Minecraft.
3. **ВАЖНО:** В настройках мира включите "Beta APIs" (Бета-версии API).

## 🛠 Возможности
- `!help` — показывает список всех имеющихся команд.
- `!aliases` — выводит короткие псевдонимы команд. Например, вместо `!home` можно написать `!h`.
- `!sethome`, `!home`, `!deletehome`, `!homelist` — система точек дома.
- `!creative`, `!spectator`, `!survival` — система более простой и быстрой смены режима игры (вместо `!creative` можно написать скорочено `!c`).
- `!troll`, `!trollall` — подкинь друзей высоко вверх.
- `!nightvision` — выдача ночного зрения себе навсегда.
- `!gift` — случайный подарок, который можно забирать раз в 30 минут.
- `!getposition` — показывает точные координаты и измерение. Можно легко линкануть друзьям.
